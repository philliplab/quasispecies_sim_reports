1 Basic homology based fitness

1.1 Description

A first basic report exploring the effects of assigning fitness based on homology.

Start with an ancestor of only As and then run two comparisons:

  • One against a comparator that is only As and Xs - lets say 20% A - using the current fitness selection tools this will not work since the ancestor will die. Hack around it by assigning the first couple of generations perfect fitness. (currently the first 63 rows)
  • One against a similat comparator, but with the As replaced with Cs
